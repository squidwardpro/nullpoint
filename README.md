# Nullpoint Calculator

A calculator app.

---

## Features

- Basic operations: `+`, `-`, `*`, `/`, `%`, `^`
- Extras: square roots, percentages, memory, history, dark mode
- Fun: emotional support, philosophy mode, verbose explanations

---

## Quick Start

```bash
git clone https://github.com/nullpoint/calculator.git
cd calculator
npm install
npm start
```

Visit `http://localhost:3000`

---

## Sample Usage

- `2 + 2` → `4`
- `45.67 * 0.20` → `9.134`
- `0 / 0` → `NaN`, plus a message to cheer you up

---

## Config

Customize with `.calculatorrc` in your home directory:

```json
{
  "precision": 47,
  "theme": "dark",
  "verboseMode": true
}
```
